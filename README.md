# Usage
[(Back to top)](#table-of-contents)

- **To perform prediction using the deep learning model, run the following commands in the terminal:**

  - (1). Download 

        git clone https://github.com/DLBioSynthesis/DLBioSign
        cd DLBioSign

  - (2). Environment

        conda create -n DLBioSign_env python=3.11
        conda activate DLBioSign_env
        pip install -r requirements.txt
    
  - (3). Generate

        conda activate DLBioSign_env
        python generate.py
